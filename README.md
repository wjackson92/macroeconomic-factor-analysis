\# Macroeconomic Factor Analysis



\## Overview



This project explores whether latent macroeconomic factors extracted from publicly available U.S. economic indicators can improve GDP forecasting. Using data from the Federal Reserve Economic Data (FRED) database, the project applies statistical factor analysis to reduce the dimensionality of a broad set of macroeconomic variables before evaluating their relationship with GDP growth.



\## Motivation



Macroeconomic forecasting often relies on large collections of highly correlated economic indicators. Rather than modeling each variable individually, factor analysis identifies a smaller number of latent factors that summarize underlying economic conditions. These latent factors provide a clearer grouping of what concepts drive GDP and how they are expressed through key indicators. Using these latent factors This project investigates whether these latent factors provide a useful representation of the economy for forecasting GDP.



\## Results



The analysis successfully extracted interpretable latent macroeconomic factors from a diverse set of economic indicators and incorporated them into a GDP forecasting framework. While the project demonstrates the application of factor analysis to macroeconomic forecasting, it also highlights several opportunities for improving feature selection, model specification, and forecast evaluation.



\## Future Improvements



Potential extensions include:



\- Expanding the macroeconomic feature set.

\- Comparing Factor Analysis with alternative dimensionality reduction techniques such as Principal Component Analysis (PCA).

\- Evaluating additional forecasting models and validation strategies.

\- Testing robustness across different economic regimes and forecasting horizons.

\- Comparing forecasting with latent factors to forecasting with raw indicators
